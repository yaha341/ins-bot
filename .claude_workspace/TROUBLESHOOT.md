# Диагностика: бот не отвечает на комментарии

## Проверка 1: Правило активно?

В админ-панели:
1. **📋 Правила** → найдите ваше правило
2. Должна быть зелёная галочка **✓ Активно**
3. Если написано "Выкл" → нажмите **▶ Вкл**

## Проверка 2: Кодовое слово совпадает?

- Комментарий: "хочу подарок"
- Кодовое слово в правиле: "хочу" → **✅ Сработает**
- Кодовое слово в правиле: "подарок" → **✅ Сработает**
- Кодовое слово в правиле: "приз" → **❌ Не сработает**

**Регистр не важен:** "ХОЧУ", "хочу", "Хочу" - всё работает

## Проверка 3: Media ID правильный?

1. **📋 Правила** → откройте правило
2. Посмотрите на **Media ID** (должен быть shortcode типа `ABC123xyz`)
3. Убедитесь что комментарий написан **под этим постом**

## Проверка 4: Сколько времени прошло?

Бот проверяет комментарии **каждые 7 минут** (по расписанию cron).

- Написали комментарий в 09:25 → бот проверит в 09:28, 09:35, 09:42...
- Может быть задержка до **7 минут**

## Проверка 5: Логи Vercel

**Самое важное** - проверьте логи:

1. Откройте https://vercel.com
2. Ваш проект → **Deployments**
3. Найдите **Functions** → кликните на `/api/poll`
4. Посмотрите логи - там будет видно:
   - `[poll] Starting comment processing...`
   - `[Apify] Getting comments for: ...`
   - `[poll] Found X comments`
   - Если есть ошибки - покажите мне

## Проверка 6: Cron работает?

Проверьте в Vercel:
1. **Settings** → **Cron Jobs**
2. Должна быть запись с `/api/poll`
3. Должна быть активна

Или откройте вручную:
```
https://ваш-проект.vercel.app/api/poll
```

Должен вернуть JSON:
```json
{"ok": true, "processed": 0, "rules": 1}
```

## Проверка 7: Комментарий новый?

Бот обрабатывает только **новые** комментарии после создания правила.

Если комментарий был написан **до** создания правила → бот его пропустит.

**Решение:** Напишите новый комментарий **сейчас**

1. Проверьте логи Vercel Functions `/api/poll` - скопируйте что там написано
2. Откройте правило - убедитесь что оно активно
3. Напишите **НОВЫЙ** комментарий с кодовым словом
4. Подождите 7 минут
5. Проверьте **📊 Лог** в админке - есть запись?
