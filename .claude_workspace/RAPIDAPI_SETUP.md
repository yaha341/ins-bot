# Переход на RapidAPI Instagram API

## Шаг 1: Регистрация на RapidAPI

1. Откройте https://rapidapi.com/
2. Нажмите **Sign Up** (можно через Google/GitHub)
3. Подтвердите email

## Шаг 2: Выбор Instagram API

Есть несколько вариантов:

### Вариант A: Instagram Bulk Profile Scraper (рекомендую)
https://rapidapi.com/mrngstar/api/instagram-bulk-profile-scrapper

- ✅ Free tier: 100 requests/month
- ✅ Простой и быстрый
- ✅ Получает комментарии

### Вариант B: Instagram Data
https://rapidapi.com/adityar2711-gmail/api/instagram-data1

- ✅ Free tier: 150 requests/month
- ✅ Много функций

### Вариант C: Instagram Scraper API
https://rapidapi.com/omarmhaimdat/api/instagram-scraper-api2

- ✅ Free tier: 100 requests/month

## Шаг 3: Подписка на API

1. Выберите один из API выше (рекомендую Вариант A)
2. Нажмите **Subscribe to Test**
3. Выберите **Basic (Free)** план
4. Подтвердите

## Шаг 4: Получение API ключа

1. После подписки нажмите **Code Snippets**
2. Справа вверху найдите **X-RapidAPI-Key**: `ваш_ключ_здесь`
3. Скопируйте ключ

## Шаг 5: Что делать дальше

Пришлите мне:
1. Какой API выбрали (A, B или C)?
2. Ваш RapidAPI ключ (начинается с букв/цифр)

Я перепишу код на RapidAPI и всё заработает за 5 минут!
