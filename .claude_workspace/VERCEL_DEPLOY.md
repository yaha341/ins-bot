# 🔧 Vercel не подтягивает деплой автоматически

## Решение: Manual Redeploy

### Шаг 1: Проверьте подключение GitHub
1. Откройте https://vercel.com → ваш проект
2. **Settings** → **Git**
3. Проверьте что репозиторий подключен

### Шаг 2: Manual Redeploy
1. **Deployments** (верхнее меню)
2. Нажмите **Create Deployment**
3. Или найдите последний деплой → **...** (три точки) → **Redeploy**

### Шаг 3: Или подключите auto-deploy
1. **Settings** → **Git**
2. Убедитесь что "Production Branch" = `main`
3. Включите "Automatically deploy new commits"

---

## Быстрый способ: Deploy через CLI

Если у вас установлен Vercel CLI:

```bash
npx vercel --prod
```

Это задеплоит текущий код сразу.

---

## Или можно протестировать БЕЗ деплоя

Пока Vercel не обновился, можно вручную запустить проверку:

Откройте в браузере:
```
https://ins-bot-xi.vercel.app/api/poll
```

Это запустит проверку комментариев прямо сейчас (без ожидания cron).

Что вернет:
- `{"ok": true, "processed": 0, "rules": 1}` - всё работает
- Или ошибку - покажите мне

После того как сделаете Redeploy в Vercel, cron начнет работать каждую минуту автоматически.
